5.Qual a diferença entre as propriedades textContent, innerText e innerHTML dos elementos HTML? Cite exemplos.

textContent: retorna todo o conteúdo textual de um elemento sem interpretar as tags HTML, substitui todo o conteúdo do elemento por texto simples.

innerText: retorna o texto visível ao usuário, respeitando as regras de estilo CSS.

innerHTML: retorna o conteúdo HTML completo de um elemento, incluindo tags internas, quando usada para atribuição, ele substitui o conteúdo atual por novo conteúdo HTML.
